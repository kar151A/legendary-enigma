# 🚀 Runway Calculator

A simple and intuitive tool to help founders understand how long their startup can survive based on current finances.

## 💡 Problem
Many startups fail because they run out of cash without realizing it in time. This tool provides a quick snapshot of financial runway.

## ⚙️ How It Works

Runway is calculated using:

Runway (months) = Total Cash / Monthly Burn Rate

## 🎯 Features

- 📊 Instant runway calculation
- 🎨 Visual status indicators:
  - 🟢 Safe (≥ 6 months)
  - 🟡 Warning (3–6 months)
  - 🔴 Danger (< 3 months)
- 💰 Currency formatting (INR)
- 📅 Days remaining calculation
- 💾 Saves inputs using localStorage
- 📱 Mobile-friendly design
- ⚡ Progress bar visualization

## 🎨 Design Decisions

- Used color psychology (green/yellow/red) for quick understanding
- Added subtle animation in "Danger" state to create urgency
- Kept UI minimal for fast usability
- Used localStorage to improve user experience
